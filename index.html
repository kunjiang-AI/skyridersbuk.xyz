How to Play

Sky Riders BUK features intuitive controls that are easy to learn but challenging to master. Use your keyboard to navigate your vehicle through the challenging sky tracks.

Controls:

  • W or Up Arrow key = Move forward
  • S or Down Arrow key = Move backward
  • A or Left Arrow key = Move left
  • D or Right Arrow key = Move right
  • Space = Handbrake
  • Left mouse button = Interact with in-game UI
